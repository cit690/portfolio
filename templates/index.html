My Work

Pseudoku

HTML, CSS, JavaScript

Pseudoku is a simplified, single board browser game based off of the game Sudoku, a logic based, number placement puzzle where the objective is to fill out a 9 x 9 grid so that each column, row, and 3 x 3 subgrid each contain the numbers 1 to 9.

End of Life Planner

ExpressJs, NodeJs, Mongoose, MongoDb, JavaScript, HTML, CSS

The End of Life Planner allows a user to create and edit their end of life plans with an easy to use form based application. This application gives users the opportunity to have their wishes for their ideal end of life out in the open for their loved ones to see.

Animal Feud

ReactJs, ExpressJs, NodeJs, Mongoose, MongoDb, JavaScript, HTML, MUI, CSS

A collaborative effort between myself and my two colleagues, Jeremy Underwood and Darnell Dugger, Animal Feud is a social web application intended for users to share facts and opinions about various animal related topics, and engage in light-hearted debates about said topics.

Lavndr

ReactJs, Python, Flask, Postgres, CSS

Developed by myself and Tye Webster, Lavndr is a social web application application with the intended purpose of helping those within the LGBTQA+ community to discover and build platonic connections. A work in progress that currently offers a user input profile form and direct message functionality.
